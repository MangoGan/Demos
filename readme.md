前端技术的核心在于：

    1. HTML：超文本标记语言，是用来描述网页的一种语言，定义了页面的结构，属于结构层
    2. CSS：层叠样式表，定义了如何显示页面，比如位置、颜色等等，是表现层。
    3. JavaScript：脚本语言，用于实现网页的功能，完成一些交互等等，属于行为层

所以我觉得从HTML和CSS开始学起比较好。
